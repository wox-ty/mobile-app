# Tunera

Минимальный шаблон проекта под концепт Tunera.

## Что есть

- `app/src/main/res/layout/activity_login.xml` — единственный XML-экран (логин).
- Compose-экраны: лента, карточка трека, шаринг, профиль.
- Мок-репозиторий `FakeRepo` и простые `UseCases`.

## Быстрый запуск

```powershell
./gradlew test
```

```powershell
./gradlew assembleDebug
```
